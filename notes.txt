Here are some details about the code variables and Reportlab Canvas :

Lists : 
Total Credits (Sum of all subject's credits) of subjects present in that semester, for each sem X is stored in `credit_sem_X` (element_datatype: int) where X = 1:10,
SGPA in sem X for each same student is stored in `student_sem_X` (element_datatype: float) where X=1:10
Total Number of Attempt for main examination in sem X for each same student is stored in `attempt_sem_X` (element_datatype: int) where X=1:10
Results (PASS, ATKT, FAIL) for each same student in that semester X  is stored in `result_sem_X` (element_datatype: str) where X=1:10

student_details_new.xls ( Input Excel file ) --> wb_student_details (Python variable for this excel sheet)
Student_Marks.xls (Input Excel file ) --> wb_subject (Python variable for this excel sheet)
By wb_subject , Subject Code,  Subject Name and Credits for each subject is stroed in different variable subject_code , subject_name and course_credits
subjects_grades is a list of list, where each inner list contains the grades of all students for a particular subject. 
So, subjects_grades[n] gives the list of grades of all students for the "nth" subject, and subjects_grades[n][k] gives the grade of the "kth" student in the "nth" subject.
Means "n" represents the subject index and "k" represents the student index.

If you want to generate Marksheet for ATKT Examination:
 Write ATKT ( or ATKT2 or ATKT3 and so on)  in the last column of Subject Excel sheet, in front of each student who has ATKT.
In the last column of Subject Excel sheet:
 Blank -> 1st Examination (Main) Examination attempt 
 ATKT -> 2nd Examination attempt
 ATKT2 -> 3rd Examination attempt
 ATKT3 -> 4th Examination attempt

 Either leave blank or ATKTX in Subject Excel sheet, Do not Mix like , In One subject excel ATKT2 and another have ATKT3.
